candle-gen-chroma is a Rust library implementing an image generation pipeline for the Chroma diffusion model family (base, HD, and flash variants). It provides model loading, noise scheduling via beta-distribution-based sigma schedules, and inference pipeline logic, built on top of a candle ML framework and a gen_core abstraction layer for generators.
