candle-gen is a Rust-based image generation framework that implements diffusion model pipelines, specifically targeting the Chroma model family (base, flash, and HD variants). The project provides noise scheduling (beta/sigma schedules), model configuration, and a generator interface built on top of a candle ML backend, enabling text-to-image generation with configurable classifier-free guidance and diffusion sampling strategies.
