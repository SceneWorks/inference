## Summary

candle-gen is a Rust-based image generation framework that wraps the Candle ML library to provide a Stable Diffusion XL (SDXL) inference pipeline. The project exposes a plugin/backend architecture where SDXL is registered as a named backend, handles text-to-image generation through a multi-step diffusion pipeline including CLIP text encoding, UNet denoising, and VAE decoding, and bridges Candle's error types to a common gen_core interface.
