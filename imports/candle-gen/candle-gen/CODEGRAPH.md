## Summary

candle-gen is a Rust-based image generation framework that wraps the Candle ML library to provide a pipeline for running Flux diffusion models (including Schnell and Dev variants, as well as a Flux2/Klein 9B variant). It exposes a plugin/registry architecture where model descriptors are registered and resolved as generators, handling model loading, configuration, sampling schedules, and text-to-image inference surfaces.
