## Summary

candle-gen is a Rust-based image generation framework that implements diffusion model inference, specifically providing support for the Chroma model in multiple variants (base, flash, HD). It handles model loading, configuration, noise scheduling via beta distribution-based sigma computation, and exposes a Generator interface for text-to-image generation using the Candle ML framework.
