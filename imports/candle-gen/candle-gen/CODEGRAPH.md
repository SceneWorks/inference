## Summary

candle-gen is a Rust-based image generation backend that implements a Stable Diffusion XL (SDXL) inference pipeline using the Candle ML framework. It exposes a plugin-style interface (ModelDescriptor, validate, generate) for text-to-image synthesis, downloading model weights from Hugging Face, running CLIP text encoding, a diffusion denoising loop, and VAE decoding to produce output images. It sits in a larger repository (gen-core) that defines the shared generation traits and types, with this project providing the concrete SDXL/RealVisXL backend.
