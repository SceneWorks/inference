candle-gen is a Rust-based image generation framework built on the Candle ML framework, implementing diffusion model pipelines. The candle-gen-chroma subproject specifically implements the Chroma diffusion model in three variants (base, flash, HD), handling noise scheduling via beta distribution-based sigma schedules, model configuration, and inference pipeline logic for text-to-image generation.
