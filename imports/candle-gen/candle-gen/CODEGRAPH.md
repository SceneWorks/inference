## Summary

candle-gen is a Rust-based image generation framework that wraps the Candle ML library to provide a Stable Diffusion XL (SDXL) inference pipeline. It exposes a backend-agnostic generation interface (via gen_core) and implements model loading, text encoding, latent diffusion, and VAE decoding for txt2img generation. The project is structured as a core utility crate (candle-gen) plus a model-specific crate (candle-gen-sdxl) that registers the SDXL backend.
