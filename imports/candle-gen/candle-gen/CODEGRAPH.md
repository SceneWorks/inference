## Summary

candle-gen is a Rust-based generative image model inference framework built on top of the Candle ML library. The candle-gen-chroma sub-project implements a diffusion-based image generation pipeline for 'Chroma' models (base, flash, and HD variants), including beta-schedule noise sigma computation, model loading/registration, and a sampling pipeline that converts text prompts into images via iterative denoising.
