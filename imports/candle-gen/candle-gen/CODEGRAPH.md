candle-gen is a Rust-based image generation framework built on the Candle ML library, implementing diffusion model pipelines. The candle-gen-chroma subproject specifically implements the Chroma image generation model with multiple variants (base, HD, flash), including noise scheduling via beta distribution-based sigma schedules, model configuration, and pipeline orchestration for text-to-image generation.
