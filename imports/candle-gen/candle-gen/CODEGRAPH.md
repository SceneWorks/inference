## Summary

candle-gen is a Rust-based generative AI inference project that implements image generation capabilities using the Candle ML framework. It provides a 'boogu' model backend that supports text-to-image generation and image editing (inpainting/reference-conditioned generation), loading quantized model weights from disk and managing model configuration, layer counts, and generation requests through a plugin-style Generator trait interface.
