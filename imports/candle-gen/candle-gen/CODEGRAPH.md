## Summary

candle-gen is a Rust-based image generation framework built on the Candle ML library, implementing diffusion model pipelines. The candle-gen-chroma subproject specifically implements the Chroma image generation model in three variants (base, flash, HD), providing noise scheduling via beta distribution-based sigma schedules, model configuration, and a generation pipeline that converts text prompts to images through iterative denoising.
