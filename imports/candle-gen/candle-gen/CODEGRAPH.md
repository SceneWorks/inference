candle-gen is a Rust-based image generation library that implements the Chroma diffusion model, providing three model variants (base, flash, and HD). It handles model configuration, noise scheduling via beta distribution-based sigma computation, and exposes a Generator interface for loading and running text-to-image generation using the Candle ML framework.
