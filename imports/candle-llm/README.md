# candle-llm

Reserved crate name for an upcoming **on-device LLM serving engine** — the [Candle](https://github.com/huggingface/candle) backend of a backend-neutral serving contract (cross-platform, including Windows).

This `0.0.0` release is a placeholder — the implementation is in progress. A sibling [`mlx-llm`](https://github.com/SceneWorks/mlx-llm) crate provides the Apple MLX backend.

## Status

Work in progress. Not yet usable.
