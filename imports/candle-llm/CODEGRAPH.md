candle-llm is a Rust library for LLM inference built on top of the Candle tensor framework. It provides a unified interface for loading, configuring, and running inference across multiple large language model architectures including LLaMA, Gemma, Qwen, GLM, and mixture-of-experts (MoE) variants.
